2. Description of Service

MarkView is a Chrome browser extension that provides markdown file rendering and viewing capabilities. The Service includes but is not limited to:

  • Markdown to HTML rendering with GitHub Flavored Markdown support
  • Syntax highlighting for 180+ programming languages
  • Diagram rendering (Mermaid flowcharts, sequence diagrams, Gantt charts)
  • Mathematical equation rendering (LaTeX/KaTeX)
  • Table of Contents (TOC) generation and navigation
  • Local file browsing and folder navigation
  • External file viewing (GitHub, GitLab, raw markdown URLs)
  • Customization options (themes, fonts, layout preferences)
  • Image viewer with lightbox and lazy loading
  • Live preview with auto-refresh for external files
