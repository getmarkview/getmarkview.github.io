Initial Release

Version 1.0.0

November 1, 2025

Initial Features

  • Syntax Highlighting: Support for 180+ programming languages
  • Mermaid Diagrams: Sequence diagrams, flowcharts, graphs, mindmaps, timelines, and more
  • Math Equations: LaTeX math rendering with KaTeX for perfect formulas and equations
  • Smart TOC: Auto-generated navigation with scroll spy, smooth scrolling, and collapsible sidebar
  • File Browser: Navigate local markdown files with search, filter, and collapsible folder structure
  • Image Viewer: Full-screen lightbox with gallery navigation and lazy loading
  • Auto-Refresh: Automatic updates for both local and external files
  • Theme System: Dark and light themes with automatic detection
  • Floating Action Buttons: Quick access to raw markdown view, centered layout toggle, theme switch, and scroll to top
  • GitHub-Flavored Markdown: Full support including task lists, tables, and code blocks with copy functionality
  • Highly Customizable: 9 font families, adjustable max width, line height, font size, with reset to defaults
